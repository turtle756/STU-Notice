# 대학 총학생회 웹사이트

## 프로젝트 개요
대학 총학생회를 위한 웹사이트로, 재학생들이 교내 행사와 공모전 정보를 쉽게 확인하고, 동아리 및 소모임 커뮤니티에 참여할 수 있도록 돕습니다.

## 주요 기능
1. **행사 & 공모전 페이지**: 교내 행사와 공모전 정보를 이미지와 텍스트로 한눈에 확인
2. **동아리 & 소모임 페이지**: 자율동아리, 소모임, 커뮤니티 정보와 카카오톡 오픈채팅 링크 제공

## 기술 스택
- React (UI 라이브러리)
- Vite (빌드 도구)
- React Router (페이지 라우팅)
- CSS (스타일링)

## 프로젝트 구조
```
src/
  ├── components/       # 재사용 가능한 컴포넌트
  │   └── Navigation.jsx
  ├── pages/           # 페이지 컴포넌트
  │   ├── Events.jsx
  │   └── Community.jsx
  └── App.jsx          # 메인 앱 컴포넌트
```

## 실행 방법
```bash
npm install
npm run dev
```
