# 대학 총학생회 웹사이트

## 프로젝트 개요
대학 총학생회를 위한 웹사이트로, 재학생들이 교내 행사와 공모전 정보를 쉽게 확인하고, 동아리 및 소모임 커뮤니티에 참여할 수 있도록 돕습니다.

## 주요 기능
1. **행사 & 공모전 페이지**: 교내 행사와 공모전 정보를 이미지와 텍스트로 한눈에 확인
2. **동아리 & 소모임 페이지**: 자율동아리, 소모임, 커뮤니티 정보와 카카오톡 오픈채팅 링크 제공
3. **필터 기능**: 카테고리별로 쉽게 필터링하여 원하는 정보만 확인
4. **반응형 디자인**: 데스크톱, 태블릿, 모바일 모든 기기에서 최적화

## 기술 스택

### HTML 버전 (현재 실행 중)
- HTML5 (구조)
- CSS3 (스타일링 및 반응형 디자인)
- Vanilla JavaScript (필터 기능)
- http-server (개발 서버)

### React 버전 (선택 사항)
- React (UI 라이브러리)
- Vite (빌드 도구)
- React Router (페이지 라우팅)
- CSS (스타일링)

## 프로젝트 구조

### HTML 버전
```
html/
  ├── index.html        # 행사 & 공모전 페이지
  ├── community.html    # 동아리 & 소모임 페이지
  └── styles.css        # 공통 스타일
```

### React 버전
```
src/
  ├── components/       # 재사용 가능한 컴포넌트
  │   └── Navigation.jsx
  ├── pages/           # 페이지 컴포넌트
  │   ├── Events.jsx
  │   └── Community.jsx
  └── App.jsx          # 메인 앱 컴포넌트
```

## 실행 방법

### HTML 버전 (현재 실행 중)
HTML 버전은 현재 자동으로 실행 중입니다. 화면 상단의 웹뷰에서 확인할 수 있습니다.

### React 버전 실행
```bash
npm install
npm run dev
```

## 수정 방법

### 행사 및 공모전 정보 수정
`html/index.html` 파일을 열어 다음을 수정하세요:
- 이미지 URL (placeholder 이미지를 실제 이미지로 교체)
- 행사명, 날짜, 설명
- 카테고리 (행사/공모전)

### 동아리 및 소모임 정보 수정
`html/community.html` 파일을 열어 다음을 수정하세요:
- 동아리/소모임 이름, 설명
- 참여 인원 수
- 카카오톡 오픈채팅방 링크
- 카테고리 (스포츠/학술/문화예술/취미)

### 스타일 변경
`html/styles.css` 파일에서 색상, 폰트, 레이아웃 등을 자유롭게 수정할 수 있습니다.
