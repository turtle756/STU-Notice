# 대학 총학생회 웹사이트

## 프로젝트 개요
대학 총학생회를 위한 순수 HTML 웹사이트로, 재학생들이 교내 행사와 공모전 정보를 쉽게 확인하고, 동아리 및 소모임 커뮤니티에 참여할 수 있도록 돕습니다.

## 주요 기능
1. **행사 & 공모전 페이지**: 교내 행사와 공모전 정보를 이미지와 텍스트로 한눈에 확인
2. **동아리 & 소모임 페이지**: 자율동아리, 소모임, 커뮤니티 정보와 카카오톡 오픈채팅 링크 제공
3. **필터 기능**: 카테고리별로 쉽게 필터링하여 원하는 정보만 확인
4. **반응형 디자인**: 데스크톱, 태블릿, 모바일 모든 기기에서 최적화

## 기술 스택
- **순수 HTML5** (구조)
- **CSS3** (HTML 내부 `<style>` 태그에 포함)
- **Vanilla JavaScript** (HTML 내부 `<script>` 태그에 포함)
- **외부 의존성 없음** (각 HTML 파일이 완전히 독립적으로 작동)

## 프로젝트 구조
```
html/
  ├── index.html        # 행사 & 공모전 페이지 (CSS, JavaScript 포함)
  └── community.html    # 동아리 & 소모임 페이지 (CSS, JavaScript 포함)
```

**각 HTML 파일은 완전히 독립적으로 작동합니다. 외부 파일 없이 HTML 파일만 있으면 됩니다.**

## 실행 방법
웹사이트는 현재 자동으로 실행 중입니다. 화면 상단의 웹뷰에서 확인할 수 있습니다.

HTML 파일을 브라우저에서 직접 열어도 작동합니다.

## 수정 방법

### 행사 및 공모전 정보 수정
`html/index.html` 파일을 열어 다음을 수정하세요:
- 이미지 URL (placeholder 이미지를 실제 이미지로 교체)
- 행사명, 날짜, 설명
- 카테고리 (행사/공모전)

### 동아리 및 소모임 정보 수정
`html/community.html` 파일을 열어 다음을 수정하세요:
- 동아리/소모임 이름, 설명
- 참여 인원 수
- 카카오톡 오픈채팅방 링크
- 카테고리 (스포츠/학술/문화예술/취미)

### 스타일 변경
각 HTML 파일의 `<style>` 태그 안에서 CSS를 수정할 수 있습니다.

### 버튼 클릭 기능 수정
각 HTML 파일의 `<script>` 태그 안에서 JavaScript를 수정할 수 있습니다.
